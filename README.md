# Українська (розширена+) розкладка для ОС Windows
Enhanced PLUS UA — кастомна українська розкладка для Windows, розширена додатковими символами та російськими літерами для випадкового використання, не впливаючи на основний український набір.  

Вона включає:
- російські літери в їхніх стандартних позиціях, як у звичайній російській розкладці;
- виправляє недолік стандартної розширеної української розкладки (велика "Ґ" тепер друкується при натиснутому *Caps Lock*);
- додає деякі символи, що зазвичай доступні лише в англійській розкладці.

### Для кого це?
Для:
- тих, хто використовує **українську** мову як **основну** й бажають підвищити ефективність друку;
- тих, мають російськомовне минуле, але перейшли чи переходять на українську;
- інтернет-тролів;
- будь-кого, хто знайде для себе в цьому користь.

# ПОЧАТОК РОБОТИ
Протестовано на Windows 11. Інструкції також до неї. На системах старших за Windows 10 робота не гарантується.


### ПЕРЕВІРКА СУМІСНОСТІ
Перейдіть у стандартні (системні) налаштування Windows -> Система -> Про систему. У розділі "Специфікації пристрою" дивіться на "Тип системи".

| Тип системи | Назва файлу для встановлення |
| --- | --- |
| _64-розрядна операційна система, процесор на базі архітектури x64_ | [...]_amd64.msi |
| _32-розрядна операційна система, процесор на базі архітектури x64_ | [...]_i386.msi |
| _32-розрядна операційна система, процесор на базі архітектури x32_ | [...]_i386.msi |
| _64-розрядна операційна система, процесор на базі архітектури **ARM**_ | __НЕ ПРАЦЮВАТИМЕ!!!__ |

### ПЕРЕВІРКА МОВИ
Якщо у вас система українською &mdash; можете пропустити цей пункт.

Перейдіть у стандартні (системні) налаштування Windows -> Час і мова -> Мова і регіон.  
Якщо в списку мов нема української:
1) Натисність кнопку "Додати мову";
2) Знайдіть українську у списку і натисність ліворуч знизу кнопку "Далі";
3) Відмітьте бажані прапорці й натисність кнопку "Інсталювати".

## ВСТАНОВЛЕННЯ
1) [Завантажити архів](https://github.com/MrBalker2999/Enhanced-PLUS-UA-keyboard-layout/releases) і розпакувати його;
1) Запустити потрібний файл (описано трохи вище);
2) Дочекатися появи вікні з повідомленням про вдале встановлення (може знадобитися трохи часу);
3) Натиснути "Close";
4) ___ОБОВ'ЯЗКОВО___ перезавантажити комп'ютер.
5) _РЕКОМЕНДОВАНО_ зайти у стандартні (системні) налаштування Windows -> Час і мова -> Мова і регіон -> трикрапка біля української мови -> Мовні параметри -> трикрапка біля "Українська (розширена)" -> Видалити).

## ПИТАННЯ
* *Як видалити?*  
    1) Запустити файл, що використовувався для встановлення (якщо не пам'ятаєте який, читайте вище розділ "Перевірка сумісності");
    2) Обрати "Remove the keyboard layout";
    3) Натиснути кнопку "Finish";
    4) Зайти у стандартні (системні) налаштування Windows -> Час і мова -> Мова і регіон -> трикрапка біля української мови -> Мовні параметри -> трикрапка біля "Укр. (розширена) з рос. літерами та повноцінною ґ" або "Українська (розширена+) розкладка" -> Видалити);
    5) Зайти у стандартні (системні) налаштування Windows -> Час і мова -> Мова і регіон -> трикрапка біля української мови -> Програми ->  Інстальовані програми -> трикрапка біля "Укр. (розширена) з рос. літерами та повноцінною ґ" або "Українська (розширена+) розкладка" (залежно від версії) -> Видалити);
    6) Перезавантажити комп'ютер.

* *Навіщо МЕНІ ця клавіатура?*  
Якщо ваша пріоритетна мова &mdash; українська, ви перемикатиметесь на одну мову менше, а, може, і на дві.  
_Тут ідеться про тримовний варіант (англійська, українська, російська)._

* *Схожа клавіатура вже є в мережі, чому обирати цю?*  
Ті клавіатури, що я знаходив або були переповнені функціоналом, або не працювали в мене (ОС Windows 11) і, ймовірно, не працюватимуть у вас.  
Ця &mdash; працюватиме як годинник.

* *В інструкції встановлення написано видалити "Українська (розширена)". Чи це безпечно?*  
Так, абсолютно. Ви видаляєте не мову, а тільки клавіатуру (розкладку).

* *Після оновлення Windows у мене знову з'явилася стандартна розкладка. Що робити?*  
Так працює ОС Windows і, на жаль, виправити це я не можу. Тут є 2 вирішення і обидва &mdash; милиці.
    - Перезавантажте комп'ютер. Windows не завжди все прибирає за собою після оновлення.
    - Якщо пепередній варіянт не допоміг: зайти в стандартні (системні) налаштування Windows -> Час і мова -> Мова і регіон -> трикрапка біля української мови -> Мовні параметри -> Додати клавіатуру -> "Українська" (імовірно, вам треба буде додати саме її, а не розширену) -> трикрапка біля "Українська" (так, це та, що була щойно встановлена) -> Видалити.

* *У назві файлу встановлення є "amd", але в мене процесор Intel. Чи працюватиме це на моєму комп'ютері?*  
Так, працюватиме. amd64 &mdash; це: 64-бітна x86-архітектура, створена AMD, повністю сумісна з Intel 64. Тобто буде працювати і на Ryzen, і на Intel Core, і навіть на Xeon. У Windows це зазвичай зветься як x64, x86-64 або amd64. Це — одне й те саме.
